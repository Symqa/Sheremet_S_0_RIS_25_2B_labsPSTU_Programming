## 3. Теоретические сведения

Библиотека STL (Standard Template Library) предоставляет набор обобщённых алгоритмов, реализованных в виде шаблонов функций. Алгоритмы работают с итераторами, поэтому могут применяться к любым контейнерам, поддерживающим соответствующий тип итераторов.

### 3.1. Основные категории алгоритмов

- **Немодифицирующие операции**: `find`, `find_if`, `count`, `count_if`, `search` и др.
- **Модифицирующие операции**: `copy`, `replace`, `replace_if`, `remove`, `remove_if`, `transform`, `unique` и др.
- **Алгоритмы сортировки**: `sort`, `partial_sort`, `stable_sort`, `binary_search`, `lower_bound`, `upper_bound`.
- **Алгоритмы работы с множествами**: `merge`, `set_union`, `set_intersection`, `set_difference`.

### 3.2. Функциональные объекты и предикаты

Функциональный объект – это класс, перегружающий оператор `()`. Предикат – функция или функциональный объект, возвращающий `bool`. Предикаты используются для задания критериев сравнения или отбора элементов. Примеры стандартных предикатов: `less<T>`, `greater<T>`, `equal_to<T>`.

### 3.3. Адаптеры функций

- **Связыватели** (`bind1st`, `bind2nd`) – превращают бинарную функцию в унарную, фиксируя один аргумент.
- **Отрицатели** (`not1`, `not2`) – инвертируют результат предиката.
- **Адаптеры методов** (`mem_fun`, `mem_fun_ref`) – позволяют использовать методы классов в алгоритмах.

В современном C++ вместо связывателей и отрицателей предпочтительнее использовать лямбда-выражения. В данной работе применяются как лямбда-функции, так и явно определённые структуры-предикаты.

## 4. Описание класса `Pair`

Класс `Pair` взят из лабораторной работы №3. Он хранит два числа: `int first` и `double second`. Для использования в алгоритмах STL в классе перегружены необходимые операторы:

- Операции сравнения (`<`, `>`, `==`, `!=`) – по ключевому полю `first`.
- Арифметические операции (`+`, `-`, `*`, `/`) – поэлементные.
- Операторы ввода/вывода (`>>`, `<<`) – дружественные функции.

**Листинг `Pair.h` (основные фрагменты):**

```cpp
#ifndef PAIR_H
#define PAIR_H
#include <iostream>

class Pair {
private:
    int first;
    double second;
public:
    Pair() : first(0), second(0) {}
    Pair(int f, double s) : first(f), second(s) {}
    
    int getFirst() const { return first; }
    double getSecond() const { return second; }
    void setFirst(int f) { first = f; }
    void setSecond(double s) { second = s; }
    
    // Операции сравнения (по first)
    bool operator<(const Pair& other) const { return first < other.first; }
    bool operator>(const Pair& other) const { return first > other.first; }
    bool operator==(const Pair& other) const {
        return first == other.first && second == other.second;
    }
    bool operator!=(const Pair& other) const { return !(*this == other); }
    
    // Поэлементные арифметические операции
    Pair operator+(const Pair& other) const;
    Pair operator-(const Pair& other) const;
    Pair operator*(const Pair& other) const;
    Pair operator/(const Pair& other) const;
    
    friend std::istream& operator>>(std::istream&, Pair&);
    friend std::ostream& operator<<(std::ostream&, const Pair&);
};

#endif
```

Реализация арифметических операций опущена для краткости (она тривиальна).

// Глобальная переменная для операции деления (используется в for_each)
Pair divisor;
void dividePair(Pair& p) {
    if (divisor.getFirst() != 0 && divisor.getSecond() != 0)
        p = p / divisor;

// Предикат для поиска по ключу

### 5.2. Задача 1. Вектор

```cpp
void task1_vector() {
    cout << "\n========== Задача 1: Вектор ==========\n";
    int n;
    cout << "Количество элементов: "; cin >> n;
    vector<Pair> v;
    for (int i = 0; i < n; ++i) {
        Pair p; cin >> p;
        v.push_back(p);
    }
    printVector(v, "Исходный вектор:");

    // Сортировка по возрастанию и убыванию
    sort(v.begin(), v.end(), [](const Pair& a, const Pair& b) { return a.getFirst() < b.getFirst(); });
    printVector(v, "По возрастанию first:");
    sort(v.begin(), v.end(), greater<Pair>());
    printVector(v, "По убыванию first:");

    // Поиск по ключу
    double key;
    cout << "Ключ для поиска: "; cin >> key;
    auto it = find_if(v.begin(), v.end(), EqualFirst(key));
    if (it != v.end()) cout << "Найден: " << *it << endl;
    else cout << "Не найден\n";

    // Задание 3: вставка минимального на позицию
    auto minIt = min_element(v.begin(), v.end());
    Pair minElem = *minIt;
    cout << "Минимальный элемент: " << minElem << endl;
    int pos; cout << "Позиция для вставки: "; cin >> pos;
    v.insert(v.begin() + pos, minElem);
    printVector(v, "После вставки минимума:");

    // Задание 4: удаление элементов меньше среднего first
    double sum = 0;
    for (auto& p : v) sum += p.getFirst();
    double avg = sum / v.size();
    cout << "Среднее first = " << avg << endl;
    v.erase(remove_if(v.begin(), v.end(), LessThanAverage(avg)), v.end());
    printVector(v, "После удаления меньших среднего:");

    // Задание 5: деление каждого на максимальный
    auto maxIt = max_element(v.begin(), v.end());
    divisor = *maxIt;
    cout << "Максимальный элемент (делитель): " << divisor << endl;
    for_each(v.begin(), v.end(), dividePair);
    printVector(v, "После деления на максимум:");
}
```

### 5.3. Задача 2. Очередь (`queue`)

Так как `queue` не предоставляет итераторов, для применения алгоритмов очередь преобразуется в вектор, а затем обратно.

```cpp
void task2_queue() {
    cout << "\n========== Задача 2: Очередь ==========\n";
    int n; cout << "Количество элементов: "; cin >> n;
    queue<Pair> q;
    for (int i = 0; i < n; ++i) {
        Pair p; cin >> p;
        q.push(p);
    }
    // Извлечение в вектор
    vector<Pair> v;
    while (!q.empty()) {
        v.push_back(q.front());
        q.pop();
    }
    printVector(v, "Содержимое очереди:");

    // Сортировка и поиск (аналогично вектору)
    sort(v.begin(), v.end());
    printVector(v, "После сортировки по возрастанию:");
    sort(v.begin(), v.end(), greater<Pair>());
    printVector(v, "По убыванию:");

    double key; cout << "Ключ для поиска: "; cin >> key;
    auto it = find_if(v.begin(), v.end(), [key](const Pair& p) { return p.getFirst() == key; });
    if (it != v.end()) cout << "Найден: " << *it << endl;
    else cout << "Не найден\n";

    // Задание 3
    auto minIt = min_element(v.begin(), v.end());
    Pair minElem = *minIt;
    int pos; cout << "Позиция для вставки минимума: "; cin >> pos;
    v.insert(v.begin() + pos, minElem);
    printVector(v, "После вставки минимума:");

    // Задание 4
    double sum = 0; for (auto& p : v) sum += p.getFirst();
    double avg = sum / v.size();
    v.erase(remove_if(v.begin(), v.end(), [avg](const Pair& p) { return p.getFirst() < avg; }), v.end());
    printVector(v, "Удалены меньшие среднего:");

    // Задание 5
    auto maxIt = max_element(v.begin(), v.end());
    divisor = *maxIt;
    for_each(v.begin(), v.end(), dividePair);
    printVector(v, "После деления на максимум:");

    // Обратно в очередь
    for (auto& p : v) q.push(p);
    cout << "Очередь восстановлена.\n";
}
```

### 5.4. Задача 3. Множество с дубликатами (`multiset`)

Для `multiset` сортировка задаётся при создании (компаратор). Операции вставки/удаления выполняются напрямую.

```cpp
void task3_multiset() {
    cout << "\n========== Задача 3: Мультимножество ==========\n";
    int n; cout << "Количество элементов: "; cin >> n;
    multiset<Pair> ms;   // по умолчанию less<Pair> (сравнение по first)
    for (int i = 0; i < n; ++i) {
        Pair p; cin >> p;
        ms.insert(p);
    }
    cout << "Исходное мультимножество:\n";
    for (auto& p : ms) cout << p << " ";
    cout << endl;

    // "Сортировка" по убыванию созданием нового с компаратором greater
    multiset<Pair, greater<Pair>> ms_desc(ms.begin(), ms.end());
    cout << "По убыванию first:\n";
    for (auto& p : ms_desc) cout << p << " ";
    cout << endl;

    // Поиск по ключу
    double key; cout << "Ключ для поиска: "; cin >> key;
    auto it = find_if(ms.begin(), ms.end(), [key](const Pair& p) { return p.getFirst() == key; });
    if (it != ms.end()) cout << "Найден: " << *it << endl;
    else cout << "Не найден\n";

    // Для операций вставки на позицию и удаления по условию перейдём к вектору
    vector<Pair> v(ms.begin(), ms.end());

    // Задание 3
    auto minIt = min_element(v.begin(), v.end());
    Pair minElem = *minIt;
    int pos; cout << "Позиция для вставки минимума: "; cin >> pos;
    v.insert(v.begin() + pos, minElem);
    ms.clear();
    ms.insert(v.begin(), v.end());

    // Задание 4
    double sum = 0; for (auto& p : ms) sum += p.getFirst();
    double avg = sum / ms.size();
    for (auto it = ms.begin(); it != ms.end(); ) {
        if (it->getFirst() < avg) it = ms.erase(it);
        else ++it;
    }
    cout << "После удаления меньших среднего:\n";
    for (auto& p : ms) cout << p << " ";
    cout << endl;

    // Задание 5
    auto maxIt = max_element(ms.begin(), ms.end());
    divisor = *maxIt;
    multiset<Pair> ms2;
    for (auto& p : ms) {
        if (divisor.getFirst() != 0 && divisor.getSecond() != 0)
            ms2.insert(p / divisor);
    }
    ms = ms2;
    cout << "После деления на максимум:\n";
    for (auto& p : ms) cout << p << " ";
    cout << endl;
}
```

## 7. Контрольные вопросы

1. **Какие категории алгоритмов существуют в STL?**  
   Немодифицирующие, модифицирующие, алгоритмы сортировки, алгоритмы работы с множествами и пирамидами.

2. **Что такое функциональный объект?**  
   Класс, в котором перегружен оператор `()`. Используется для передачи пользовательских операций в алгоритмы.

3. **Что такое предикат?**  
   Функциональный объект или обычная функция, возвращающая `bool`. Применяется для задания условий отбора или сравнения.

4. **Какие адаптеры функций вы знаете?**  
   Связыватели (`bind2nd`, `bind1st`), отрицатели (`not1`, `not2`), адаптеры указателей на функции (`ptr_fun`), адаптеры методов (`mem_fun`, `mem_fun_ref`). В современном C++ их роль выполняют лямбда-выражения и `std::bind`.

5. **Почему алгоритмы STL не выполняют проверку границ?**  
   Для повышения производительности. Пользователь должен гарантировать, что передаваемые итераторы образуют корректный диапазон.

6. **Какие алгоритмы использованы в работе для удаления элементов?**  
   `remove_if` (в сочетании с `erase` для вектора) и прямой `erase` с итератором для мультимножества.
