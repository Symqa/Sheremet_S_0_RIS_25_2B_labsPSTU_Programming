### 2. Описание пользовательского класса

Класс `Pair` инкапсулирует два числа и предоставляет интерфейс для работы с ними, включая перегруженные операции сравнения, вычитания константы, умножения на целое, а также ввод/вывод в стандартные и файловые потоки.

**Заголовочный файл `Pair.h`:**
```cpp
#pragma once
#include <iostream>
#include <fstream>
using namespace std;

class Pair {
private:
    int first;
    double second;
public:
    Pair();
    Pair(int, double);
    Pair(const Pair&);
    ~Pair();
    Pair& operator=(const Pair&);

    // Операции сравнения
    bool operator==(const Pair&) const;
    bool operator!=(const Pair&) const;
    bool operator<(const Pair&) const;
    bool operator>(const Pair&) const;
    bool operator<=(const Pair&) const;
    bool operator>=(const Pair&) const;

    // Вычитание константы
    Pair operator-(int) const;        // целая константа – уменьшается first
    Pair operator-(double) const;    // вещественная константа – уменьшается second

    // Умножение на целое (для масштабирования)
    Pair& operator*=(int factor);

    // Потоковый ввод-вывод
    friend ostream& operator<<(ostream&, const Pair&);
    friend istream& operator>>(istream&, Pair&);
    friend fstream& operator<<(fstream&, const Pair&);
    friend fstream& operator>>(fstream&, Pair&);
};
```

### 3. Перегруженные операции пользовательского класса

Все операции реализованы в файле `Pair.cpp`. Приведём ключевые реализации.

**Операции сравнения**
```cpp
bool Pair::operator==(const Pair& p) const {
    return (first == p.first) && (second == p.second);
}
bool Pair::operator!=(const Pair& p) const {
    return !(*this == p);
}
bool Pair::operator<(const Pair& p) const {
    if (first < p.first) return true;
    if (first > p.first) return false;
    return second < p.second;
}
bool Pair::operator>(const Pair& p) const {
    return p < *this;
}
// <= и >= определены через отрицание > и <
```
Сравнение производится лексикографически: сначала по полю `first`, затем по `second`.

**Вычитание константы**
```cpp
Pair Pair::operator-(int val) const {
    Pair res(*this);
    res.first -= val;
    return res;                     // создаётся новый объект, исходный не меняется
}
Pair Pair::operator-(double val) const {
    Pair res(*this);
    res.second -= val;
    return res;
}
```
Если аргумент целый, уменьшается первое число; если вещественный (с дробной частью) — второе. Вызов `obj - 7` подходит под `int`, `obj - 2.5` — под `double`. Фактическое наличие дробной части определяет компилятор по типу литерала.

**Умножение на целое (оператор `*=`)**
```cpp
Pair& Pair::operator*=(int factor) {
    first *= factor;
    second *= factor;
    return *this;
}
```
Эта операция добавлена для удобства выполнения задания «увеличить в два раза».

**Потоковые операции**
```cpp
ostream& operator<<(ostream& out, const Pair& p) {
    out << p.first << ":" << p.second;
    return out;
}
istream& operator>>(istream& in, Pair& p) {
    cout << "first (int)? "; in >> p.first;
    cout << "second (double)? "; in >> p.second;
    return in;
}
fstream& operator<<(fstream& fout, const Pair& p) {
    fout << p.first << "\n" << p.second << "\n";
    return fout;
}
fstream& operator>>(fstream& fin, Pair& p) {
    fin >> p.first;
    fin >> p.second;
    return fin;
}
```
Операции для `fstream` записывают/читают поля как отдельные строки, что обеспечивает простой текстовый формат хранения.

### 4. Определение функций для работы с файлом

Все функции размещены в заголовочном файле `file_work.h` и используют вспомогательный файл `"temp"` для изменения состава записей.

**Создание файла**
```cpp
int make_file(const char* f_name) {
    fstream stream(f_name, ios::out | ios::trunc);
    if (!stream) return -1;
    int n;
    Pair p;
    cout << "N? "; cin >> n;
    for (int i = 0; i < n; ++i) {
        cin >> p;
        stream << p;
    }
    stream.close();
    return n;
}
```
Открывает файл для записи со стиранием, записывает N объектов, введённых пользователем.

**Печать файла**
```cpp
int print_file(const char* f_name) {
    fstream stream(f_name, ios::in);
    if (!stream) return -1;
    Pair p;
    int i = 0;
    while (stream >> p) {
        cout << p << "\n";
        ++i;
    }
    stream.close();
    return i;
}
```

**Удаление записей из интервала [k1, k2]**
```cpp
int del_interval(const char* f_name, const Pair& k1, const Pair& k2) {
    fstream temp("temp", ios::out);
    fstream stream(f_name, ios::in);
    if (!stream) return -1;
    Pair p;
    int deleted = 0;
    while (stream >> p) {
        if (stream.eof()) break;
        if (!(p >= k1 && p <= k2)) {
            temp << p;
        } else {
            ++deleted;
        }
    }
    stream.close(); temp.close();
    remove(f_name);
    rename("temp", f_name);
    return deleted;
}
```
Алгоритм: читаем исходный файл, копируем во временный только те записи, которые НЕ попадают в интервал. Затем заменяем оригинал временным файлом.

**Увеличение записей с заданным значением в два раза**
```cpp
int double_value(const char* f_name, const Pair& val) {
    fstream temp("temp", ios::out);
    fstream stream(f_name, ios::in);
    if (!stream) return -1;
    Pair p;
    int changed = 0;
    while (stream >> p) {
        if (stream.eof()) break;
        if (p == val) {
            p *= 2;
            ++changed;
        }
        temp << p;
    }
    stream.close(); temp.close();
    remove(f_name);
    rename("temp", f_name);
    return changed;
}
```
Используется оператор `==` для поиска нужных записей и `operator*=` для их удвоения.

**Добавление K записей в начало файла**
```cpp
int add_begin(const char* f_name, const vector<Pair>& new_records) {
    fstream temp("temp", ios::out);
    fstream stream(f_name, ios::in);
    if (!stream) return -1;
    for (const auto& rec : new_records) {
        temp << rec;
    }
    Pair p;
    while (stream >> p) {
        if (stream.eof()) break;
        temp << p;
    }
    stream.close(); temp.close();
    remove(f_name);
    rename("temp", f_name);
    return new_records.size();
}
```
Сначала во временный файл записываются новые объекты, затем все старые. После замены файла новые записи оказываются в начале.

### 5. Функция main()

Главная функция организует меню для вызова всех описанных выше возможностей.

```cpp
#include "Pair.h"
#include "file_work.h"
#include <iostream>
#include <vector>
using namespace std;

int main() {
    Pair p1, p2;
    int k, c;
    char file_name[30];

    do {
        cout << "\n1. Make file";
        cout << "\n2. Print file";
        cout << "\n3. Delete records from interval [k1, k2]";
        cout << "\n4. Double records with given value";
        cout << "\n5. Add K records to beginning of file";
        cout << "\n0. Exit\n";
        cin >> c;

        switch (c) {
        case 1:
            cout << "file name? "; cin >> file_name;
            k = make_file(file_name);
            if (k < 0) cout << "Can't make file\n";
            break;
        case 2:
            cout << "file name? "; cin >> file_name;
            k = print_file(file_name);
            if (k == 0) cout << "Empty file\n";
            if (k < 0) cout << "Can't read file\n";
            break;
        case 3:
            cout << "file name? "; cin >> file_name;
            cout << "Enter k1:\n"; cin >> p1;
            cout << "Enter k2:\n"; cin >> p2;
            if (p2 < p1) swap(p1, p2);
            k = del_interval(file_name, p1, p2);
            if (k < 0) cout << "Can't read file\n";
            else cout << "Deleted " << k << " records\n";
            break;
        case 4:
            cout << "file name? "; cin >> file_name;
            cout << "Enter value to double:\n"; cin >> p1;
            k = double_value(file_name, p1);
            if (k < 0) cout << "Can't read file\n";
            else if (k == 0) cout << "No such records\n";
            else cout << "Changed " << k << " records\n";
            break;
        case 5: {
            cout << "file name? "; cin >> file_name;
            int K;
            cout << "K? "; cin >> K;
            vector<Pair> new_pairs;
            for (int i = 0; i < K; ++i) {
                cout << "Enter pair " << i + 1 << ":\n";
                Pair pp;
                cin >> pp;
                new_pairs.push_back(pp);
            }
            k = add_begin(file_name, new_pairs);
            if (k < 0) cout << "Can't read file\n";
            else cout << "Added " << k << " records\n";
            break;
        }
        case 0: break;
        default: cout << "Wrong choice\n";
        }
    } while (c != 0);

    return 0;
}
```

### 7. Ответы на контрольные вопросы

1. **Что такое поток?**  
   Поток – абстрактное понятие, обозначающее последовательность байтов, передаваемых между программой и устройством (файл, клавиатура, экран, память). Поток не зависит от конкретного устройства, а для повышения скорости часто используется буферизация.

2. **Какие типы потоков существуют?**  
   Стандартные (cin, cout, cerr, clog), строковые (stringstream), файловые (fstream, ifstream, ofstream).

3. **Какую библиотеку надо подключить при использовании стандартных потоков?**  
   `<iostream>`.

4. **Какую библиотеку надо подключить при использовании файловых потоков?**  
   `<fstream>`.

5. **Какую библиотеку надо подключить при использовании строковых потоков?**  
   `<sstream>`.

6. **Какая операция используется при выводе в форматированный поток?**  
   Перегруженная операция сдвига влево `<<`.

7. **Какая операция используется при вводе из форматированных потоков?**  
   Перегруженная операция сдвига вправо `>>`.

8. **Какие методы используются при выводе в форматированный поток?**  
   `put(char)`, `write(const char*, int)`.

9. **Какие методы используется при вводе из форматированного потока?**  
   `get()`, `get(char&)`, `read(char*, int)`, `getline(char*, int, char)`.

10. **Какие режимы для открытия файловых потоков существуют?**  
    `in`, `out`, `trunc`, `app`, `ate`, `binary`.

11. **Какой режим используется для добавления записей в файл?**  
    `ios::app` (открыть для дозаписи в конец) либо комбинация `ios::out | ios::app`.

12. **Какой режим (комбинация режимов) используется в конструкторе `ifstream file("f.txt")`?**  
    `ios::in` (по умолчанию для ifstream).

13. **Какой режим (комбинация режимов) используется в конструкторе `fstream file("f.txt")`?**  
    По умолчанию `ios::in | ios::out`, однако для выполнения операций необходимо явно задавать режим.

14. **Какой режим (комбинация режимов) используется в конструкторе `ofstream file("f.txt")`?**  
    `ios::out | ios::trunc` (по умолчанию для ofstream).

15. **Каким образом открывается поток в режиме `ios::out | ios::app`?**  
    `fstream stream("file.txt", ios::out | ios::app);` или `ofstream stream("file.txt", ios::app);`

16. **Каким образом открывается поток в режиме `ios::out | ios::trunc`?**  
    `ofstream stream("file.txt", ios::out | ios::trunc);` или просто `ofstream stream("file.txt");`

17. **Каким образом открывается поток в режиме `ios::out | ios::in | ios::trunc`?**  
    `fstream stream("file.txt", ios::out | ios::in | ios::trunc);`

18. **Каким образом можно открыть файл для чтения?**  
    `ifstream stream("file.txt");` или `fstream stream("file.txt", ios::in);`

19. **Каким образом можно открыть файл для записи?**  
    `ofstream stream("file.txt");` (со стиранием) или `fstream stream("file.txt", ios::out | ios::trunc);`

20. **Привести примеры открытия файловых потоков в различных режимах.**  
    - Чтение: `ifstream in("data.txt");`  
    - Запись со стиранием: `ofstream out("data.txt");`  
    - Дозапись: `ofstream out("data.txt", ios::app);`  
    - Чтение и запись с удалением старого содержимого: `fstream fs("data.txt", ios::in | ios::out | ios::trunc);`

21. **Привести примеры чтения объектов из потока.**  
    ```cpp

    while (in >> p) { cout << p; }   // чтение с помощью перегруженного >>
    // или с использованием getline для строк
    ```

22. **Привести примеры записи объектов в поток.**  
    ```cpp

    Pair p(10, 2.5);

    ```

23. **Сформулировать алгоритм удаления записей из файла.**  
    1. Открыть исходный файл для чтения.  
    2. Создать и открыть временный файл для записи.  
    3. Читать объекты из исходного файла; если объект не подлежит удалению, записать его во временный файл.  
    4. Закрыть оба файла.  
    5. Удалить исходный файл (`remove()`).  
    6. Переименовать временный файл в имя исходного (`rename()`).

24. **Сформулировать алгоритм добавления записей в файл.**  
    *В конец:* открыть файл в режиме `ios::app` и записать новые объекты.  
    *В произвольное место:*  
    1. Открыть исходный файл для чтения.  
    2. Создать временный файл для записи.  
    3. Копировать во временный файл записи до места вставки.  
    4. Записать новые объекты.  
    5. Дописать оставшиеся записи из исходного файла.  
    6. Закрыть файлы, удалить оригинал, переименовать временный файл.

25. **Сформулировать алгоритм изменения записей в файле.**  
    1. Открыть исходный файл для чтения, создать временный для записи.  
    2. Читать объекты из исходного файла; если запись нужно изменить, записать во временный файл изменённый объект, иначе — исходный.  
    3. Закрыть оба файла.  
    4. Удалить исходный файл, переименовать временный.  
    Для подтверждения изменения можно запросить согласие пользователя (как в примере с `change_file`).
