- `max_withdrawal_amount` – максимальная сумма, которую можно снять (`double`).

Необходимо реализовать конструкторы, деструктор, методы установки/получения значений полей, операции снятия (`withdraw`) и пополнения (`deposit`), а также функцию вывода состояния (`show`).


## 2. Описание класса

Класс `ATM` инкапсулирует данные банкомата и предоставляет публичный интерфейс для работы с ними.

**Заголовочный файл `ATM.h`:**
```cpp
#ifndef ATM_H
#define ATM_H

class ATM {
private:
    int identificator;
    double balance;
    double max_withdrawal_amount;
public:
    ATM();                           // конструктор без параметров
    ATM(int id, double b, double max_amount); // конструктор с параметрами
    ATM(ATM &obj);                  // конструктор копирования
    ~ATM();                         // деструктор

    void setId(int id);
    void setBalance(int b);
    void setMaxWithdraw(int max_amount);
    int getId() const;
    double getBalance() const;
    double getMaxAmount() const;
    void withdraw(double amount);
    void deposit(double amount);
    void show() const;
};

#endif
```


## 3. Определение компонентных функций

Все методы реализованы в файле `ATM.cpp`. Ниже приведены описания каждого метода.

**Конструктор без параметров**
```cpp
ATM::ATM() {
    setId(rand()%100+1);
    setBalance(rand()%100);
    setMaxWithdraw(rand()%100+1);
}
```
Создаёт объект с нулевыми или начальными значениями полей (через сеттеры для проверки).

**Конструктор с параметрами**
```cpp
ATM::ATM(int id, double b, double max_amount) {
    setId(id);
    setBalance(b);
    setMaxWithdraw(max_amount);
}
```
Инициализирует поля переданными значениями с проверкой корректности.

**Конструктор копирования**
```cpp
ATM::ATM(ATM &obj) {
    setId(obj.getId());
    setBalance(obj.getBalance());
    setMaxWithdraw(obj.getMaxAmount());
}
```
Создаёт новый объект как копию существующего, используя значения, полученные через геттеры.

**Деструктор**
```cpp
ATM::~ATM() {
    cout << endl << "Объект с идентификатором " << this->getId() << " удалён" << endl;
}
```
Выводит сообщение при уничтожении объекта, что позволяет отследить моменты вызова деструктора.

**Сеттеры (модификаторы)**
```cpp
void ATM::setId(int id) {
    if (id > 0) this->identificator = id;
    else cout << "Ошибка: идентификатор должен быть > 0" << endl;
}

void ATM::setBalance(int b) {          // параметр int, но поле double
    if (b >= 0) this->balance = b;
    else cout << "Ошибка: баланс должен быть >= 0" << endl;
}

void ATM::setMaxWithdraw(int max_amount) {
    if (max_amount > 0) this->max_withdrawal_amount = max_amount;
    else cout << "Ошибка: максимальная сумма снятия должна быть > 0";
}
```
Каждый сеттер проверяет допустимость значения: идентификатор > 0, баланс ≥ 0, максимальная сумма снятия > 0. При ошибке выводится сообщение, значение не изменяется.

**Геттеры (селекторы)**
```cpp
int ATM::getId() const { return this->identificator; }
double ATM::getBalance() const { return this->balance; }
double ATM::getMaxAmount() const { return this->max_withdrawal_amount; }
```
Возвращают текущие значения полей.

**Метод `withdraw` (снятие средств)**
```cpp
void ATM::withdraw(double amount) {
    int balance = this->getBalance();
    if (amount > 0 && balance - amount >= 0 && amount <= this->getMaxAmount()) {
        setBalance(balance - amount);
        cout << "Списание: -" << amount << " | Остаток на счёте: " << this->getBalance();
    } else {
        cout << "Ошибка: невозможно выполнить операцию списания." << endl;
    }
}
```
Проверяет, что сумма положительна, не превышает баланс и лимит снятия. При успехе уменьшает баланс.

**Метод `deposit` (пополнение)**
```cpp
void ATM::deposit(double amount) {
    int balance = this->getBalance();
    if (amount > 0) {
        setBalance(balance + amount);
        cout << "Пополнение: +" << amount << " | Остаток на счёте: " << this->getBalance();
    } else {
        cout << "Ошибка: невозможно выполнить операцию пополнения..." << endl;
    }
}
```
Увеличивает баланс, если сумма положительна.

**Метод `show` (вывод состояния)**
```cpp
void ATM::show() const {
    cout << "==========Банкомат============" << endl;
    cout << "Идентификатор: " << this->getId() << endl;
    cout << "Баланс: " << this->getBalance() << endl;
    cout << "Максимальная сумма списания: " << this->getMaxAmount() << endl;
}
```
Выводит все атрибуты объекта в отформатированном виде.


## 4. Определение внешних функций make_atm() и print_atm()

Эти функции созданы в файле `main.cpp` для демонстрации вызова конструктора копирования.

**Функция `make_atm()` – возврат объекта из функции**
```cpp
ATM make_atm() {
    int id;
    double balance, max_withdraw;
    cout << "Введите идентификатор банкомата: ";
    cin >> id;
    cout << "Введите остаток денег в банкомате: ";
    cin >> balance;
    cout << "Введите максимальную сумму снятия: ";
    cin >> max_withdraw;
    ATM atm(id, balance, max_withdraw);
    return atm;   // здесь вызывается конструктор копирования
}
```
Создаёт локальный объект `atm` с введёнными пользователем параметрами и возвращает его по значению. При возврате компилятор создаёт временную копию, вызывая конструктор копирования.

**Функция `print_atm()` – передача объекта по значению**
```cpp
void print_atm(ATM a) {
    a.show();
}
```
Принимает объект `ATM` по значению. При вызове создаётся копия переданного аргумента, вызывается конструктор копирования. Внутри функции выводится информация об объекте.

**Полный код основной программы (`main.cpp`):**
```cpp
#include "ATM.h"
#include <iostream>
using namespace std;

ATM make_atm() {
    int id;
    double balance, max_withdraw;
    cout << "Введите идентификатор банкомата: ";
    cin >> id;
    cout << "Введите остаток денег в банкомате: ";
    cin >> balance;
    cout << "Введите максимальную сумму снятия: ";
    cin >> max_withdraw;
    ATM atm(id, balance, max_withdraw);
    return atm;
}

void print_atm(ATM a) {
    a.show();
}

int main() {
    // 1. Конструктор без параметров
    cout << "=== 1. Конструктор без параметров ===" << endl;
    ATM atm1;
    atm1.show();

    // 2. Конструктор с параметрами
    cout << "\n=== 2. Конструктор с параметрами ===" << endl;
    ATM atm2(1001, 150000.0, 50000.0);
    atm2.show();

    // 3. Конструктор копирования (инициализация другим объектом)
    cout << "\n=== 3. Конструктор копирования (atm3 = atm2) ===" << endl;
    ATM atm3 = atm2;
    atm3.show();

    // Изменение данных через сеттеры
    atm3.setId(2002);
    atm3.setBalance(75000);
    atm3.setMaxWithdraw(30000);
    cout << "После изменения данных atm3:" << endl;
    atm3.show();

    // 4. Конструктор копирования при передаче в функцию
    cout << "\n=== 4. Передача объекта в функцию (print_atm) ===" << endl;
    print_atm(atm3);

    // 5. Конструктор копирования при возврате из функции
    cout << "\n=== 5. Возврат объекта из функции (make_atm) ===" << endl;
    atm1 = make_atm();
    cout << "Полученный объект atm1:" << endl;
    atm1.show();

    // Дополнительно: операции снятия и пополнения
    cout << "\n=== Дополнительно: операции с банкоматом atm2 ===" << endl;
    atm2.withdraw(10000);
    cout << endl;
    atm2.deposit(25000);
    cout << endl;
    atm2.show();

    cout << "\n=== Завершение main, объекты будут удалены ===" << endl;
    return 0;
}
```

## 6. Ответы на контрольные вопросы

1. **Для чего нужен конструктор?**  
   Конструктор – это метод класса, предназначенный для инициализации объекта при его создании. Он вызывается автоматически и позволяет задать начальные значения полей.

2. **Сколько типов конструкторов существует в C++?**  
   Три типа: конструктор без параметров (по умолчанию), конструктор с параметрами, конструктор копирования.

3. **Для чего используется деструктор? В каких случаях деструктор описывается явно?**  
   Деструктор освобождает ресурсы, выделенные объекту (память, файлы и т.д.). Явно описывается, когда класс использует динамически выделенную память или другие ресурсы, требующие специального освобождения.

4. **Для чего используется конструктор без параметров? Конструктор с параметрами? Конструктор копирования?**  
   - Без параметров – создаёт объект с начальными значениями по умолчанию.  
   - С параметрами – инициализирует объект конкретными значениями.  
   - Копирования – создаёт новый объект как точную копию существующего.

5. **В каких случаях вызывается конструктор копирования?**  
   - При создании нового объекта и инициализации его другим объектом (`ATM a = b;`).
   - При передаче объекта в функцию по значению.
   - При возврате объекта из функции по значению.

6. **Перечислить свойства конструкторов.**  
   - Имя совпадает с именем класса.
   - Не имеют возвращаемого значения.
   - Вызываются автоматически при создании объекта.
   - Могут быть перегружены.
   - Не наследуются.
   - Не могут быть `const`, `virtual` или `static`.
   - Параметры могут иметь значения по умолчанию.

7. **Перечислить свойства деструкторов.**  
   - Имя начинается с тильды (`~`) и совпадает с именем класса.
   - Не имеет параметров и возвращаемого значения.
   - Не может быть перегружен.
   - Вызывается автоматически при уничтожении объекта.
   - Не наследуется.
   - Может быть виртуальным.
   - Не может быть `const` или `static`.

8. **К каким атрибутам имеют доступ методы класса?**  
   Ко всем атрибутам класса независимо от спецификаторов доступа (private, public, protected).

9. **Что представляет собой указатель `this`?**  
   Неявный указатель, передаваемый в каждый нестатический метод класса. Содержит адрес текущего объекта, для которого вызван метод.

10. **Какая разница между методами, определёнными внутри класса и вне класса?**  
    Методы, определённые внутри класса, по умолчанию считаются встроенными (`inline`). Определённые вне класса требуют указания имени класса с оператором разрешения области видимости (`::`).

11. **Какое значение возвращает конструктор?**  
    Конструктор не возвращает никакого значения (даже `void`).

12. **Какие методы создаются по умолчанию?**  
    Если программист не объявил, компилятор автоматически создаёт: конструктор без параметров, конструктор копирования, деструктор и оператор присваивания.

13. **Какое значение возвращает деструктор?**  
    Деструктор не возвращает значения.

14. **Дано описание класса**  
    ```cpp
    class Student {
        string name;
        int group;
    public:
        student(string, int);
        student(const student&);
        ~student();
    };
    ```  
    Какой метод отсутствует в описании класса?  
    Отсутствует конструктор без параметров.

15. **Какой метод будет вызван при выполнении `student *s; s = new student;`?**  
    Конструктор без параметров (если он определён) или сгенерированный по умолчанию.

16. **Какой метод будет вызван при выполнении `student s("Ivanov", 20);`?**  
    Конструктор с параметрами.

17. **Какие методы будут вызваны при выполнении `student s1("Ivanov", 20); student s2 = s1;`?**  
    Для `s1` – конструктор с параметрами, для `s2` – конструктор копирования.

18. **Какие методы будут вызваны при выполнении `student s1("Ivanov", 20); student s2; s2 = s1;`?**  
    Для `s1` – конструктор с параметрами, для `s2` – конструктор без параметров, затем оператор присваивания (не конструктор).

19. **Какой конструктор будет использоваться при передаче параметра в функцию `void print(student a) { a.show(); }`?**  
    Конструктор копирования (объект передаётся по значению).

20. **Класс описан: `class Student { string name; int age; public: void set_name(string); ... }; Student p;` Каким образом можно присвоить новое значение атрибуту `name` объекта `p`?**  
    С помощью метода-модификатора: `p.set_name("Новое имя");`. Прямой доступ `p.name` невозможен, так как `name` находится в private-секции (по умолчанию для `class`).
```
